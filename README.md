<h1>SGE - Sistema de Gerenciamento de Estoque</h1>

O SGE (Sistema de Gerenciamento de Estoque) é uma aplicação desktop desenvolvida em Java, com integração ao MySQL, 
criada para auxiliar empresas no controle eficiente de seus produtos, movimentações e estoque.
O objetivo do sistema é centralizar o gerenciamento de mercadorias, oferecendo uma interface intuitiva para cadastro, 
consulta e acompanhamento das operações de entrada e saída de produtos, contribuindo para uma administração mais organizada e precisa.

Funcionalidades: 

-- Cadastro de produtos
-- Edição e exclusão de produtos
-- Controle de entrada de mercadorias
-- Controle de saída de produtos
-- Atualização automática do estoque
-- Pesquisa de produtos por nome ou código
-- Consulta do estoque disponível
-- Histórico de movimentações
-- Cadastro de categorias
-- Relatórios de estoque

Tecnologias utilizadas: 

-- Java
-- Java Swing
-- Hibernate
-- MySQL
-- NetBeans
-- Git e GitHub

Arquitetura do projeto: 

O sistema foi desenvolvido seguindo boas práticas de organização do código, utilizando:

-- DTO (Data Transfer Object) para transporte de dados.
-- DAO (Data Access Object) para acesso ao banco de dados.
-- Hibernate para comunicação com o MySQL.
-- Swing para construção da interface gráfica.
-- Programação Orientada a Objetos (POO).

Objetivo: 

Este projeto tem como finalidade aplicar conceitos de desenvolvimento de software, banco de dados, orientação a objetos e arquitetura em camadas, 
simulando um sistema utilizado em ambientes empresariais para gerenciamento de estoque.
Além do aprendizado, o SGE faz parte do meu portfólio como desenvolvedor Java, demonstrando conhecimentos em desenvolvimento desktop, 
persistência de dados, integração com banco de dados relacionais e organização de projetos utilizando boas práticas da Engenharia de Software.
